## B. BRANDING STUDIO CG.STUDIO

### Positionnement visuel
**Concept directeur** : "Narrative Engineering" — rigueur technique + ambition créative. Pas corporate, pas startup. Studio arthouse avec crédentiels broadcast.

---

### Système couleur

| Rôle | Valeur | Usage |
|------|--------|-------|
| Background principal | `#0B0B0B` | Site, déclinaisons dark |
| Accent studio | `#002FA7` (Klein Blue) | Wordmark, CTA, highlights |
| Texte principal | `#F5F5F0` (off-white) | Corps de texte sur fond dark |
| Surface secondaire | `#1A1A1A` | Cards, sections en retrait |
| IP Miss One | À définir depuis Bible Visuelle | Matériaux M1 uniquement |
| IP Die with Peace | `#3D2B1F` (sépia chaud) | Matériaux DwP uniquement |

**Règle** : jamais plus de 3 couleurs sur un même support. Le Klein Blue est réservé aux moments d'emphase — pas de fond plein bleu.

---

### Typographie

**Wordmark** : Helvetica Neue Light ou Inter Thin.
- "CG." semi-bold, légèrement oversized
- "STUDIO" thin, même baseline, letter-spacing +200

**Titres** : Inter 300, tracking +50, uppercase, pas de gras.

**Corps** : Inter 400, 16-18px, line-height 1.7, jamais condensé.

**Règle** : 2 graisses maximum par support. Aucun serif sauf DwP key arts.

---

### Brief Wordmark CG.STUDIO

**Structure à produire** :
- Version horizontale standard : `CG · STUDIO`
- Version stacked (carré) : `CG` au-dessus de `STUDIO`
- Monogramme seul : `CG` pour watermarks vidéo

**Fichiers cibles** :
- SVG vectoriel (source)
- PNG 4K fond transparent — version dark (blanc/bleu sur transparent)
- PNG 4K fond transparent — version light (noir sur transparent)

**Outil recommandé** : Figma ou Illustrator. Si IA : Canva Magic Studio ou Looka pour draft, affinage manuel obligatoire.

## C. SIZZLE REEL STUDIO — SCRIPT 60-90 SEC

### Concept
**"CG.STUDIO — Narratives that live past the screen"**
Format : montage atmosphérique, texte à l'écran minimal, musique ambiante. Pas de talking head. Pas de voix off.

---

### Structure séquencée

**0:00 → 0:05 — BLACK / WORDMARK** *(5 sec)*
Fondu depuis noir. CG.STUDIO wordmark apparaît progressivement.
Texte overlay : *"Independent Narrative Studio. Paris."*
Son : silence ou ambient très léger.

**0:05 → 0:20 — CRÉDENTIELS BROADCAST** *(15 sec)*
Coupe rapide d'images d'archives : Roland Garros, Sky News, ARTE, Bloomberg, Eurovision.
Règle : pas de logos des chaînes, juste les images.
Texte overlay : noms seuls en blanc 300, 1 seconde chacun.
Rythme soutenu — crédibilité sans complaisance.

**0:20 → 0:40 — ŒUVRES PRIMÉES** *(20 sec)*
3 extraits de 6 sec chacun :
- PTSD : scène forte, atmosphérique (intérieur, lumière dramatique)

Overlay par extrait : titre + 1 award en petit (Chelsea FF · Best Film / Amsterdam · Best Editing / SF · First Amendment Gallery).

Miss One : key art ou logo animé → texte *"M1 and the Network Guardians · Animation Series · Ages 9-13"*
Die with Peace : plan atmosphérique (lac, brume) → texte *"Die with Peace · Drama Series · English"*
Temps : ~7 sec par IP.

**0:55 → 1:10 — CLOSER** *(15 sec)*
Plan large, espace ouvert, nature ou architecture (choix atmosphérique).
Texte centré : *"Two IPs. One studio. Stories worth making."*
Fondu vers wordmark CG.STUDIO seul.
Fade to black.
*cedricgodin.com*

### Templates de posts

**Template A — IP Reveal**
```
[Titre]. [Tagline en une ligne].

[2-3 lignes : univers, ton, enjeu — sans spoiler, sans pitch deck verbatim]

[Fait concret : format, épisodes, public cible, diffuseurs visés]

[Une phrase personnelle : pourquoi cette histoire, pourquoi maintenant]

#animation #screendevelopment #kidstv
```

**Template B — Crédentiel Story**
```
[Œuvre]. [Prix ou diffuseur]. [Année].

Ce qu'on ne voit pas : [1 vérité concrète sur le process ou la décision difficile]

[2-3 lignes max — anecdote précise, pas générale]

Ce que j'en retiens pour aujourd'hui : [lien direct avec travail actuel]
```

**Template C — Thought Leadership**
```
[Question tranchée ou affirmation contre-intuitive en ouverture — 1 ligne]

[Développement : 3-4 paragraphes courts, pas de listes, pas de bullet points]

[Conclusion personnelle — pas de leçon morale, pas de "call to action" explicite]

Votre take ?
```

**Template D — Atmosphérique / teaser visuel**
```
[Image forte en attachement — key art ou still]

[3-5 mots seulement en caption, ou une ligne poétique]

[Titre de l'IP en hashtag discret]

#cgstudio #indev
```
